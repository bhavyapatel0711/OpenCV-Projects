# OpenCV Projects

A collection of Computer Vision projects built using **Python**, **OpenCV**, and **MediaPipe**. These projects explore real-time image processing, hand tracking, pose estimation, gesture recognition, and human-computer interaction.

---

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
* Pycaw

---

## Projects

| Project                    | Description                                                    | Concepts Used                                                     |
| -------------------------- | -------------------------------------------------------------- | ----------------------------------------------------------------- |
| Gesture Controlled Volume  | Control system volume using hand gestures.                     | Hand Tracking, Distance Normalization, Human-Computer Interaction |
| Bicep Curl Tracker         | Real-time bicep curl counter using pose estimation.            | Pose Estimation, Angle Calculation, Exercise Tracking             |
| Gesture Recognition System | Recognizes 25+ single-hand and two-hand gestures in real time. | Hand Tracking, Geometry, State Machines, Temporal Smoothing       |

This repository was created to:

* Understand MediaPipe's landmark detection pipelines.

* Explore geometry-based gesture recognition techniques.
* Develop modular and reusable OpenCV systems.
